# Сайт по поверке и замене водяных счетчиков

Современный адаптивный сайт для услуги по поверке и замене водяных счетчиков в Липецке.

## 🎯 Особенности

- ✅ **Две основные услуги:**
  - Поверка счетчика — 500₽
  - Замена счетчика — 1600₽

- 📱 **Полностью адаптивный дизайн** под все устройства
- 🎨 **Современный чистый дизайн** без градиентов
- 📝 **Умная форма заявки** с автоматическим расчетом стоимости
- ✨ **Плавные анимации** и переходы
- 📞 **Автоматическая маска телефона** (+7 формат)
